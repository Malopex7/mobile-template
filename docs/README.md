# Expo + Express Monorepo Starter Template

This is a production-grade template using Expo Router (React Native), Express, MongoDB, and TypeScript.

## Quick Start

1. Ensure you have Yarn installed.
2. Run `yarn install` from root.
3. Setup environment variables (see \`.env.example\` in apps).

### Running
- Backend: `yarn dev:api`
- Mobile: `yarn start:mobile`

## Stack
- Mobile: Expo, NativeWind, Tanstack Query, Zustand
- Backend: Express, Mongoose, Socket.io, Zod Validator
- Shared: Zod schemas and TypeScript types.
