# Deployment

## Railway
1. Create a monorepo project on Railway.
2. Add the API service setting the Root Directory to `/apps/api`.
3. Set the start command to `yarn start`.
4. Supply necessary env keys (`MONGO_URI`, `JWT_SECRET`, etc.).

## Render
1. Similar setup. Define Root Directory as `/apps/api`.
2. Add Web Service.
3. Configure ENV strings securely.
